# PBExp

Jednoduchá webová stránka pre úlohu.

Súbory: `index.html`, `styles.css`, `script.js`.

Publikovať cez GitHub Pages.
# Kyberbezpečnosť — Demo šifrovania

Krátke vzdelávacie demo zobrazujúce symetrické šifrovanie (AES-GCM) a deriváciu kľúča z hesla (PBKDF2) pomocou Web Crypto API.

Ako spustiť lokálne:

1. Otvorte [index.html](index.html) v prehliadači dvojklikom.
2. Ak prehliadač blokuje niektoré funkcie pri `file://`, spustite lokálny server (príklad s Python):

```powershell
python -m http.server 8000
# potom otvorte http://localhost:8000
```

Poznámka: Demo slúži na vzdelávacie účely — neodporúča sa ukladať reálne citlivé dáta do nezabezpečených prostredí.
